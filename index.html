    Guide

    使用说明

    新手必读
    1. 导入书籍

      点击顶部「导入书籍」，选择 PDF、EPUB、TXT 或 MD 文件。安卓手机把本应用安装到主屏幕后，也可以在文件管理器里对文件点「分享」选择 iStone Reader 直接导入。

    2. 开始听读

      在「朗读控制」里选好语音和语速，点「开始朗读」。支持暂停/继续、上一句/下一句、定时关闭。语速拖动后立即生效，不用重新开始。

    3. 书架与进度

      导入过的书会自动保存到「本机书架」，听到哪里也会自动记录。下次打开应用会直接恢复上次的书和位置，不需要重新导入。

    4. 扫描版 PDF

      没有文字层的扫描版 PDF 会自动启用 OCR 文字识别，首次识别需要加载识别模型，会慢一些，请耐心等待页面顶部的进度提示。

    常见问题

    点了朗读没有声音？

    先点「测试发声」确认设备能出声。常见原因：手机媒体音量为零、iPhone 侧面静音拨片打开、或者页面是在微信等 App 内置浏览器里打开的——内置浏览器对网页朗读限制较多，建议点右上角菜单选「在浏览器中打开」，用系统自带的 Chrome、Edge 或 Safari 使用。

    锁屏或切到别的应用后朗读停了？

    网页朗读使用手机系统语音，部分手机锁屏后会被系统中断。应用在朗读时会尽量保持屏幕常亮；如果被打断，回到本页面会自动从当前句继续。长时间听书建议保持页面在前台，或把手机充着电、屏幕常亮使用。

    语音列表为什么每台设备不一样？

    朗读声音来自设备系统和浏览器，不同手机、不同浏览器能用的声音不同。安卓手机可以在系统设置的「文字转语音 / TTS」里安装或更新语音引擎来获得更好的中文声音；iPhone 可在「辅助功能 → 朗读内容 → 声音」里下载更高质量的语音。

    扫描版识别慢、或识别结果不准？

    首次识别需要加载几 MB 的识别模型，之后会快很多。清晰、摆正、单栏排版的扫描件效果最好；拍照倾斜、分辨率低、双栏、手写体或水印严重的页面识别效果会明显下降。可以在「色调与识别」里切换识别语言（默认中英混合，会按内容自动优化）。

    怎么导入微信里收到的文件？

    微信聊天文件需要先保存出来再导入：安卓在微信里打开文件 → 「···」→「保存到手机」，然后在系统文件管理的「Download/WeiXin」里把文件「分享」给 iStone Reader（需先安装到主屏幕）；iPhone 在微信里选「用其他应用打开 → 存储到"文件"」，再用「导入书籍」从"文件"里选取。点顶部「导入微信文件」按钮可查看带步骤的完整指引。

    我的书和进度存在哪里？会上传吗？

    不会上传。文件解析、识别和朗读全部在你自己的设备上完成，书架和进度只保存在本机浏览器里。注意：清除浏览器数据（或无痕模式）会清空书架；换设备或换浏览器不会同步。

    如何安装到桌面 / 主屏幕？

    电脑 Chrome / Edge：地址栏右侧的安装图标，或浏览器菜单里的「安装应用」。安卓 Chrome：右上角菜单 →「添加到主屏幕 / 安装应用」。iPhone Safari：底部分享按钮 →「添加到主屏幕」。安装后可像普通应用一样从桌面打开。

    遇到其他问题，可以先点「测试发声」和「加载演示书」排查是设备问题还是文件问题。

    导入微信里收到的文件

    微信聊天里的文件存放在微信自己的空间里，网页无法直接读取，需要先「保存出来」再导入。请按你的手机类型操作：

    安卓手机

    1. 在微信里点开那个文件，点右上角「···」，选择「保存到手机」（部分机型叫「存储到手机」）。
    2. 最快的方式：打开系统「文件管理」，进入「下载 / Download / WeiXin」文件夹，长按文件选「分享 / 发送」，在分享列表里选择 iStone Reader，会直接导入并开始解析。
      前提：已把本应用「添加到主屏幕 / 安装应用」。如果分享列表里没有 iStone Reader，请先安装。
    3. 备选方式：回到本页点「导入书籍」，在文件选择器里进入「下载（Download）」下的 WeiXin 文件夹选取文件。

    iPhone / iPad

    1. 在微信里点开那个文件，点右上角「···」，选择「用其他应用打开」，然后选「存储到"文件"」并保存。
    2. 回到本页点「导入书籍」，在弹出的"文件"界面里打开「最近项目」或刚才保存的位置，点选文件即可。

    提示：请不要在微信内置浏览器里使用本应用（限制较多且无法朗读），建议用系统浏览器打开并添加到主屏幕。
